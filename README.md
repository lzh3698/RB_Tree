从零实现的自平衡二叉搜索树，深入理解高级数据结构

一、项目简介

	一个完整的C++红黑树实现，包含插入、删除、查找等核心操作，严格维护红黑树的五条性质，并提供四种遍历方式。

二、实现功能

	1、插入操作 (insert)：支持红黑树的标准插入，包含颜色调整和旋转
	
	2、删除操作 (erase)：支持节点删除，包含删除后的平衡修复
	
	3、迭代器支持：
	
		① iterator：普通迭代器，支持++操作遍历树
		
		② const_iterator：常量迭代器，用于只读访问
		
	4、查询操作：
	
		① max：获取树中最大元素
		
		② min：获取树中最小元素
		
		③ find：查找元素
		
	5、状态检查：
	
		① empty：检查树是否为空
		
		② size：获取树中元素个数
		
	6、四种旋转平衡操作
	
	7、四种遍历方式（前序、中序、后序、层序）
	
	8、内存安全，无内存泄漏
	
三、实现难点

	1、删除操作的四种修复情况
	
	2、复杂的指针与递归调试

四、算法复杂度

	操作	时间复杂度	空间复杂度
	
	插入	O(log n)	  O(1)
	
	删除	O(log n)	  O(1)
	
	查找	O(log n)	  O(1)
	
五、遍历功能

	支持四种遍历方式，便于调试和观察树结构：
	
	1、前序遍历：根节点 → 左子树 → 右子树
	
	2、中序遍历：左子树 → 根节点 → 右子树（有序输出）
	
	3、后序遍历：左子树 → 右子树 → 根节点
	
	4、层序遍历：按层次从上到下，从左到右
